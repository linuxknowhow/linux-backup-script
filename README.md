# linux-backup-script

## What is it?

This is a linux backup script. It's used for backuping:

- Folders
- MySQL databases
- Cron entries

## Where does it upload backups?

- Locally
- AWS S3

## Killer features:

- YAML config
- Archivers with password protection
- Chaining of archivers
- GPG encryption

## How to run it:

```
php /path/to/script/backup.php --create
```
